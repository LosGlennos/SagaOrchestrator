# Saga Orchestrator Pattern - Slidev Presentation

This directory contains the Slidev presentation for the Saga Orchestrator Pattern tech talk.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The presentation will be available at `http://localhost:3030`

### Build for Production

```bash
npm run build
```

### Export to PDF

```bash
npm run export
```

## Presentation Structure

The presentation covers:

1. **Introduction** - What is the Saga Orchestrator Pattern?
2. **Architecture** - System overview and technology stack
3. **Saga Flow** - Step-by-step breakdown of the booking saga
4. **Compensation** - How compensation works when failures occur
5. **Event Sourcing** - Event store architecture and state reconstruction
6. **Technical Implementation** - Message broker, state management, error handling
7. **Demo Scenarios** - Success and failure scenarios
8. **Key Learnings** - Best practices and design decisions
9. **Live Demo** - Interactive demonstration

## Navigation

- **Space** or **Arrow Right** - Next slide
- **Arrow Left** - Previous slide
- **F** - Fullscreen
- **G** - Overview mode
- **O** - Presenter mode

## Customization

Edit `slides.md` to customize the presentation content.
